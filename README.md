# TodoApp
Creating a simple TodoApp using the MERN Stack to apply my first learnings in the field of web development. 

This is a simple web application in which I will be applying my initial learnings of the MERN Stack. 
A todo application is an application which can be used by the user to make and manage their daily tasks or future goals. In this application I aim to provide the user a simple user interface to add and/or remove tasks in their to do lists.
The application can be used by anyone just by logging in through their e-mail id and password.
This repository is maintained by me only as this is a simple project and their is no need for more than one person maintaining the codebase.

Hope you enjoy using the todo app and have a good productive day.
